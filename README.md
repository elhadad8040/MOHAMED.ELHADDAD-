# تطبيق الكوثر (QuranAI)

تطبيق ذكي لتحليل وتحسين تلاوة القرآن الكريم باستخدام تقنيات الذكاء الاصطناعي.

## نظرة عامة

تطبيق الكوثر هو تطبيق متخصص في تحليل تلاوة القرآن الكريم وتقديم اقتراحات للتحسين. يستفيد التطبيق من تقنيات الذكاء الاصطناعي ومعالجة الصوت لاكتشاف الأخطاء وتقييم جودة القراءة بناءً على قواعد التجويد.

## المميزات الرئيسية

- تسجيل التلاوة مباشرة داخل التطبيق
- تحليل شامل للتلاوة وكشف الأخطاء
- تشخيص مخارج الحروف والقواعد التجويدية
- تقديم اقتراحات للتحسين وشرح للأخطاء
- واجهة مستخدم سهلة الاستخدام ومصممة خصيصًا للغة العربية

## هيكل المشروع

```
lib/
  ├── blocs/             # طبقة إدارة الحالة باستخدام BLoC
  ├── models/            # نماذج البيانات
  ├── screens/           # شاشات التطبيق
  ├── services/          # خدمات التطبيق (تسجيل الصوت، API)
  ├── widgets/           # مكونات واجهة المستخدم القابلة لإعادة الاستخدام
  └── main.dart          # نقطة الدخول الرئيسية للتطبيق

assets/
  ├── images/            # الصور
  ├── icons/             # الأيقونات
  ├── audio/             # ملفات الصوت
  └── fonts/             # الخطوط
```

## متطلبات التشغيل

- Flutter SDK: 2.19.0 أو أحدث
- Dart: 2.19.0 أو أحدث
- مساحة تخزين كافية لملفات الصوت
- ميكروفون للتسجيل

## طريقة التثبيت

1. استنساخ المشروع:
```
git clone https://github.com/yourusername/quranai_flutter_app.git
```

2. الانتقال إلى دليل المشروع:
```
cd quranai_flutter_app
```

3. تثبيت التبعيات:
```
flutter pub get
```

4. تشغيل التطبيق:
```
flutter run
```

## المكتبات المستخدمة

- flutter_bloc: لإدارة حالة التطبيق
- flutter_sound: لتسجيل الصوت وتشغيله
- permission_handler: لإدارة أذونات الوصول
- http: للاتصال بخدمات API
